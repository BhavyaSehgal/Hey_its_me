Cattle Detection Occlusion Problem

This project focuses on cattle detection using various deep learning models, including YOLOv7, RetinaNet with EfficientNet and ResNet50 backbones, and Mask R-CNN. By employing image tiling techniques, such as 2x1 and 3x3 tiling, the project enhances detection accuracy, particularly for small and hidden cattle. Results show that YOLOv7 outperforms the other models in precision, demonstrating effective implementation for agricultural applications.
